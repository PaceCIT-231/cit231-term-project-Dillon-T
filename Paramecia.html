Gomu Gomu no Mi
The Gomu Gomu no Mi is a Paramecia-type Devil Fruit[4] that grants the user's body the properties of rubber, effectively making them a Rubber Human
Awa Awa no Mi
The Awa Awa no Mi is a Paramecia type Devil Fruit that allows the user to emit and control soap and bubbles that can not only clean off dirt but can also "clean off" power
Ito Ito no Mi
The Ito Ito no Mi is a Paramecia-type Devil Fruit that allows the user to create and manipulate strings, making the user a String Human
